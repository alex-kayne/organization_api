# organization_api

Тестовое задание: REST API для управления организационной структурой компании.

Проект реализован на FastAPI и PostgreSQL. API позволяет создавать подразделения, добавлять сотрудников, получать дерево подразделений, перемещать подразделения и удалять их в режимах `cascade` или `reassign`.

## Стек

- Python 3.12
- FastAPI
- SQLAlchemy Async ORM
- PostgreSQL
- Alembic
- Docker / Docker Compose
- Poetry

## Возможности API

### Подразделения

- Создание подразделения
- Получение подразделения с сотрудниками и дочерними подразделениями
- Обновление названия и родительского подразделения
- Удаление подразделения:
  - `cascade` — удалить подразделение вместе с дочерними подразделениями и сотрудниками
  - `reassign` — удалить подразделение, а сотрудников и дочерние подразделения перенести в другое подразделение

### Сотрудники

- Создание сотрудника внутри подразделения
- Валидация существования подразделения при создании сотрудника
